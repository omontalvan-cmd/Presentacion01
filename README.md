# Proyecto Módulo 1 – Fundamentos de Python

## Descripción
Aplicación desarrollada en Streamlit que integra conceptos fundamentales de programación:
variables, condicionales, estructuras de datos, funciones, programación funcional y POO.

## Estructura
- app.py
- requirements.txt
- README.md
- dmc.png
- retorno.py

## Instrucciones de ejecución
1. Instalar dependencias:
   pip install - requirements.txt
2. Ejecutar la aplicación:
   streamlit run app.py
